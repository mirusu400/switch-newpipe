# Testing Checklist

## 목적

이 문서는 실기에서 무엇을 확인해야 하는지 빠르게 체크하기 위한 목록이다. 문제 발생 시 로그는 `sdmc:/switch/switch_newpipe.log`를 본다.

## 지금까지 구현된 것

- 실제 YouTube 홈 / 검색 / 구독 / 상세 / 채널 / 연관 추천
- 로그인 기반 추천 홈 1차 연결
- watch-next 기반 연관 영상 파서
- 재생목록 / 댓글 파서
- 외부 쿠키 import 기반 로그인 세션 저장
- 최근 시청 / 즐겨찾기 저장
- Switch 전용 mpv 플레이어 루프
- 재생 로딩 오버레이와 상태 문구
- 설정 저장
  - 시작 탭
  - 기본 홈 카테고리
  - 재생 품질 정책
  - 짧은 영상 숨기기

## 테스트 전 준비

- 최신 `switch_newpipe.nro` 배포 여부 확인
- 로그인 테스트가 필요하면 `switch_newpipe_auth.txt`를 `sdmc:/switch/`에 배치
- 이전 로그가 헷갈리면 `switch_newpipe.log`를 지우고 시작

## 부팅 / 기본 동작

- 앱이 부팅 직후 튕기지 않는지
- 홈 / 검색 / 구독 / 라이브러리 / 설정 탭이 모두 보이는지
- 탭 이동과 포커스 이동이 끊기지 않는지
- 목록에서 `Y` 상세 화면이 정상 표시되는지

## 홈 / 검색

- 홈 탭에서 카드 목록이 실제 유튜브 데이터로 보이는지
- 로그인 세션이 있을 때 `추천` 탭이 개인화 feed 느낌으로 바뀌는지
- `Y`로 홈 카테고리 순환이 되는지
- `X` 새로고침이 정상인지
- 검색 탭에서 아무 키워드나 넣었을 때 결과가 나오는지
- 짧은 영상 숨기기 설정을 켠 뒤 홈 / 검색 결과가 달라지는지

## 상세 / 채널 / 연관

- 목록에서 `Y`로 상세 화면이 열리는지
- 상세에서 설명 / 메타데이터가 보이는지
- 상세에서 `X` 채널 업로드가 열리는지
- 상세에서 `Y` 연관 추천이 열리는지
- 상세에서 `LB` 메뉴가 뜨는지
- 상세에서 `LB -> 재생목록`이 열리는지
- 상세에서 `LB -> 댓글`이 열리는지
- 연관 추천이 단순 검색 느낌이 아니라 실제 watch-next 느낌의 목록인지

## 재생

- 목록에서 `A`로 바로 재생되는지
- 로딩 progress circle과 상태 문구가 보이는지
- 재생 후 `A` pause / resume 동작이 맞는지
- `B`로 플레이어 종료 후 앱으로 복귀하는지
- `위 / 아래` 볼륨이 동작하는지
- 첫 재생 직후 OSD가 자동으로 잠깐 보이는지
- `A` pause / resume 때 OSD 상태 문구가 바뀌는지
- `위 / 아래` 볼륨 조절 시 OSD 볼륨 바가 반응하는지
- `X` 또는 `Y`로 OSD 고정 표시 토글이 되는지
- 긴 영상에서 초반 몇 초 뒤 멈추거나 fatal error가 없는지

## 재생 품질 설정

- 설정 탭에서 `재생 품질` 변경 가능 여부
- `표준 720p`에서 재생 성공 여부
- `호환성 우선`에서 재생 성공 여부
- `데이터 절약`에서 재생 성공 여부
- 설정을 바꾼 뒤 체감 화질이나 로딩 특성이 달라지는지

## 설정 persistence

- 설정 탭에서 `시작 탭` 변경 후 앱 재실행 시 반영되는지
- `기본 홈 카테고리` 변경 후 홈 첫 진입이 바뀌는지
- `짧은 영상 숨기기` 변경 후 재실행 뒤에도 유지되는지
- `X` 기본값 복원이 동작하는지
- `sdmc:/switch/switch_newpipe_settings.json` 파일이 생성되는지

## 로그인 / 구독

- `switch_newpipe_auth.txt` import가 되는지
- 구독 탭에서 로그인 세션 상태가 보이는지
- 구독 목록이 실제 계정 기준으로 뜨는지
- 구독 목록에서도 `A` 재생, `Y` 상세가 되는지
- 앱 재실행 후 저장된 세션이 유지되는지
- 홈 `추천` 탭도 로그인 반영 상태로 바뀌는지

## 라이브러리

- 재생한 영상이 최근 시청에 쌓이는지
- 상세에서 `RB` 즐겨찾기 토글이 되는지
- 라이브러리에서 최근 시청 / 즐겨찾기 전환이 되는지
- `RB`로 현재 섹션 비우기가 동작하는지
- `sdmc:/switch/switch_newpipe_library.json` 파일이 생성되는지

## 문제 발생 시 기록할 것

- 어떤 탭 / 어떤 버튼 / 어떤 영상에서 문제였는지
- 재생 품질 설정이 무엇이었는지
- 로그인 세션 사용 중이었는지
- 로그 마지막 20~50줄
- 가능하면 재현 절차를 짧게 3줄 이내로 정리
